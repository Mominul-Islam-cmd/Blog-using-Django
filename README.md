# DjangoMediumClone

A Medium-style blogging platform built using **Django** and **JavaScript**. Users can create, edit, delete, and read blog posts, interact with comments, and experience a dynamic, responsive interface similar to Medium.

---

## Features

- **User Authentication:** Sign up, login, and logout functionality using Django's authentication system.
- **Create & Manage Posts:** Users can create, edit, and delete blog posts.
- **Comments & Likes:** Readers can comment on posts and like them.
- **Dynamic Frontend:** Interactive UI using JavaScript for real-time updates and smooth user experience.
- **Responsive Design:** Works seamlessly on desktop and mobile devices.
- **Search & Filter:** Search posts by title, author, or content.

---

## Technologies Used

- **Backend:** Django, Django REST Framework (optional for API)
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite (default) or PostgreSQL
- **Authentication:** Django built-in authentication system
- **Deployment:** Can be deployed on Heroku, Render, or any cloud platform
